# Инструкция по развертыванию инфраструктуры

1. Установите Terraform, если еще не установлен.
2. Склонируйте этот репозиторий.
3. Настройте доступ к облачному провайдеру, например, Яндекс.Облако.
4. Запустите команду:
   ```bash
   terraform init
   terraform plan
   terraform apply
